# 🎓 Optern - Personalized Learning Platform

Optern is a web application designed to help students navigate their learning journey with personalized roadmaps and curated resources.

## 🚀 Features
- **Firebase Authentication:** Secure login for students.
- **Firestore Integration:** Dynamic content delivery for Skills and User Progress.
- **Dashboard:** Real-time tracking of learning progress.
- **Skills Catalog:** 10 categories of industry-relevant skills.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6 Modules)
- **Backend:** Firebase Auth & Firestore
- **Deployment:** GitHub Pages (Coming Soon)
